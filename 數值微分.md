# 數值微分

函數的微分是指對函數的局部變化的一種線性描述。微分可以近似地描述當函數自變量的取值作足夠小的改變時，函數的值如何改變。[^1] 而就圖形上來看，微分的意義便是取函數中某一點的切線斜率(slope of tangent line)。

## 一次導數(first derivative)

考慮一個連續且可微的函數(continuous and differtiable function)，如果我們想知道函數 $f(x)$ 在 $x = x_0$ 時的導數(derivative)，我們可以寫成 $f'(x = x_0)$，其計算方式如下：

$$
 f'(x_0) = \frac{df}{dx} \vert_{x=x_0} = \lim_{h \rightarrow 0} \frac{f(x_0+h) - f(x_0)}{h}.
$$

就實務上而言，$h$ 必須盡可能的小，小到接近 $0$。以下面的函數為例，給定以下的二次函數：

$$
f(x) = - \frac{1}{2}x^2 + 20
$$

其一次導數為

$$
f'(x) = -x
$$

代表導數的值會隨著 $x$ 的不同而有所改變。我們可以用 $\texttt{julia}$ 的 $\verb|Symbolics|$ 套件[^2] 進行微分。

```julia
using Interact, WebIO, Plots
using Symbolics

@variables x
f(x) = -(1/2)x^2 + 20
sec(x, a, h) = ((f(a+h)-f(a))/h)

Symbolics.derivative(f(x),x) |> display
```

用圖示呈現的話大概是像這樣：

![](https://i.imgur.com/JOQbCrT.gif)

```julia
a = -4.0

@manipulate for h = 1e-9:0.1:9 
    f_sec(x::Float64) = sec(x,a,h) # hjw
    
    plot(size=(600,400))
    plot!(f, xlims=[-12,12], ylims = [8,25] , aspect_ratio = 1, label="f(x)", linewidth=2) # main curve
    plot!(f_sec, label="secant line")  
    scatter!([a], [f(a)], label=false)      # intersection a
    scatter!([a+h], [f(a+h)], label=false)  # intersection b
    
    plot!([a,a+h], [f(a)<f(a+h) ? f(a) : f(a+h), f(a)<f(a+h) ? f(a) : f(a+h)], ls=:dash, color=:darkred, label=false)
    plot!([f(a)<f(a+h) ? a+h : a, f(a)<f(a+h) ? a+h : a], [f(a),f(a+h)], ls=:dash, color=:darkred, label=false)
    quiver!([a+h/2], [f(a)<f(a+h) ? f(a)-3 : f(a+h)-3], quiver=([0],[3]), color=:darkred)
    quiver!([f(a)<f(a+h) ? a+h+3 : a-3], [(f(a)+f(a+h))/2-2], quiver=([f(a)<f(a+h) ? -3 : 3],[2]), color=:darkred)
    annotate!([a+h/2], [f(a)<f(a+h) ? f(a)-3.5 : f(a+h)-3.5], text("h=$(round(h, digits=4))", 9))
    annotate!([f(a)<f(a+h) ? a+h+3 : a-3], [(f(a)+f(a+h))/2-2.5], text("f(x+h)-f(x) = $(round(f(a+h)-f(a), digits=4)) ",9))
    
    annotate!(a-1, f(a), text("($a,$(f(a)))", :orange, :right, 9))
    annotate!(a+h+1, f(a+h), text("$(round.((a+h,f(a+h)), digits=4))", :purple, :left, 9))
  
    quiver!([a+h+1-3], [f_sec(a+h+1)+2], quiver=([3],[-2])   )
    annotate!(a+h+1-3, f_sec(a+h+1)+2.5, text("slope=$( round((f(a+h)-f(a))/h, digits=6) )", :red, 9)) 
end
```
